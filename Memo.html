Memorandum :

Memos are concise written communications within an organization. They are typically used to convey important information, make announcements, request actions, or provide updates to colleagues or team members.

Types of Memos

  1. Informative Memo
    1. Progress Report Memo
    2. Financial Report Memo
    3. Incident Report Memo
  2. Policy Memo
    1. Employee Handbook Update Memo
    2. Security Policy Memo
    3. Technology Usage Policy Memo
  3. Request Memo
    1. Equipment Purchase Request Memo
    2. Training Request Memo
    3. Time Off Request Memo
  4. Meeting Memo
    1. Departmental Meeting Memo
    2. Project Kickoff Meeting Memo
  5. Summary Memo
    1. Report Summary Memo
    2. Conference Summary Memo
  6. Instructional Memo
    1. Safety Procedures Memo
    2. Software Installation Instructions Memo
  7. Follow-up Memo
    1. Action Items Follow-up Memo
    2. Proposal Follow-up Memo
